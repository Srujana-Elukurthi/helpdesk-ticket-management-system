# Help Desk Ticket Management System

The Helpdesk Ticket Management System is a Full Stack Web Application built in Flask with a MongoDB Backend that is used to effectively manage Helpdesk Tickets.
It allows users to create accounts, login, create support tickets, and allows Administrators access to view and change ticket statuses.

---

## Features:
- User Registration and Login
- Role Based Access (Admin / User)
- Ticket Creation and Tracking
- Admin Dashboard to Manage Tickets
- Update Ticket Status (Open / In Progress / Closed)
- Clean Responsive UI

---

## Tech Stack:
- Backend – Python (Flask)
- Database – MongoDB
- Frontend - HTML, CSS, JavaScript
- Tools - Git, Github, MongoDB Compass

## How to Run the Project

1. Start MongoDB:
./mongod --dbpath ~/mongodb-data

2. Run the Flask application:
python3 app.py

3. Open in browser:
http://127.0.0.1:5000/login
